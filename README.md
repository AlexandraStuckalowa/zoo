 # Лабораторная работа № 9 "Работа с Big Data"

## Цель: 

Разработать простое приложение для работы с большими данными

## Описание нашего веб-приложения:

Проект представляет собой веб-приложение для анализа данных о посещаемости зоопарка.

Пользователь загружает CSV-файл с данными, после чего приложение рассчитывает основные показатели, строит графики и обучает модель машинного обучения для предсказания дней с высокой посещаемостью.


## Используемые технологии

- Python
- Flask
- Pandas
- Matplotlib
- Scikit-learn

## Датасет

В проекте используются датасеты `zoo_visits` и `zoo_visits_big`

В датасете есть следующие признаки:

- `date` - дата
- `weekday` - день недели
- `weather` - погода
- `temperature` - температура
- `is_holiday` - выходной день
- `ticket_price` - цена билета
- `event` - наличие мероприятия
- `visitors` - количество посетителей
- `high_attendance` - высокая посещаемость

День считается днем с высокой посещаемостью, если количество посетителей больше 2500. Если посетителей 2500 или меньше, посещаемость считается обычной.

## Возможности приложения

Приложение выполняет следующие действия:

1. Загружает данные из CSV-файла.
2. Рассчитывает основную статистику по датасету.
3. Строит графики:
   - количество дней по типу погоды;
   - среднее количество посетителей по дням недели;
   - распределение обычной и высокой посещаемости.
4. Обучает модель машинного обучения.
5. Показывает точность модели.
6. Показывает пример предсказания.

## Обучение модели

Модель предсказывает значение признака `high_attendance`

Для обучения используются признаки:

- день недели;
- погода;
- температура;
- выходной день;
- цена билета;
- наличие мероприятия.

В качестве модели используется `RandomForestClassifier`

## Как запустить веб-приложение

1. Установить зависимости:

```bash
python -m pip install -r requirements.txt
```

2. Запустить приложение: 

```bash
python app.py
```

3. Открыть в браузере:

```bash
http://127.0.0.1:5000
```

4. На странице выбрать CSV-файл и нажать кнопку: "Проанализировать"

После этого приложение покажет статистику, графики, результат обучения модели и пример предсказания.
